## 1) Introduction to Programming

### Definition / Explanation
- **Programming** is the process of writing instructions for a computer to solve a problem.
- A program is a set of **ordered steps** that the computer follows.
- Programming is used to automate tasks, solve problems, and build software.

### Key Concepts
- **Input**: data given to the computer.
- **Processing**: the computer works on the data.
- **Output**: the result produced.
- **Algorithm**: a step-by-step method to solve a problem.
- **Debugging**: finding and fixing errors.

### Example
Problem: Add two numbers.

Pseudo-code:
```text
1. Take two numbers
2. Add them
3. Display the sum
```

Python-style example:
```python
a = 10
b = 5
print(a + b)
```

### Important Points for MCQs
- A computer follows **instructions exactly**.
- Programming is different from **typing random code**; it needs logic.
- Algorithm comes **before** coding.

### Exam Focus
- Likely questions:
	- Define programming.
	- What is an algorithm?
	- Arrange input, processing, output in order.
- Common mistakes:
	- Confusing program with algorithm.
	- Thinking a computer can understand vague instructions.

## 12) Introduction to Loops

### Definition / Explanation
- A loop repeats a set of instructions multiple times.
- Loops are used when the same action must happen again and again.

### Key Concepts
- **Repetition**
- **Iteration**
- **Count-controlled loop**
- **Condition-controlled loop**

### Example
```python
for i in range(5):
		print("Hello")
```
- This prints Hello 5 times.

### Important Points for MCQs
- Loops save time and reduce repeated code.
- Loops are useful in animations, games, and data processing.
- Repetition is the main idea behind loops.

### Exam Focus
- Likely questions:
	- What is a loop?
	- Why are loops useful?
- Common mistakes:
	- Writing repeated code manually instead of using loops.
	- Confusing loop with selection/decision.

## 13) More about Loops

### Definition / Explanation
- Loops can run a fixed number of times or until a condition changes.
- Common loop types in programming include `for` and `while`.

### Key Concepts
- **Initialization**
- **Condition check**
- **Update step**
- **Termination**

### Example
```python
i = 1
while i <= 3:
		print(i)
		i = i + 1
```

### Important Points for MCQs
- A loop needs a stopping condition.
- Without a proper update, a loop may run forever.
- `for` loop is often used when the number of repetitions is known.

### Exam Focus
- Likely questions:
	- Difference between `for` and `while` loops.
	- What causes an infinite loop?
- Common mistakes:
	- Forgetting to update the loop variable.
	- Missing the loop condition.

## 14) Solution to Looping Problem

### Definition / Explanation
- The looping problem is the problem of repeating steps many times without writing the same code again and again.
- Loops solve this problem by using a compact structure.

### Key Concepts
- **Code reduction**
- **Reusability**
- **Less error-prone**
- **Cleaner programs**

### Example
Without loop:
```python
print("A")
print("A")
print("A")
print("A")
```

With loop:
```python
for i in range(4):
		print("A")
```

### Important Points for MCQs
- Loops reduce repeated statements.
- Loops make code shorter and easier to maintain.
- A single loop can replace many duplicate lines.

### Exam Focus
- Likely questions:
	- How do loops solve repetition problems?
	- Compare repeated code with loop-based code.
- Common mistakes:
	- Thinking loops are only for counting.
	- Forgetting that loops improve readability.
