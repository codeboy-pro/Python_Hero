# 🔹 1. Introduction to Programming

### ✅ Definition

* Programming = **giving instructions to a computer** to perform tasks.

### 🔑 Key Concepts

* Computer follows **exact instructions only**
* Instructions must be:

  * Clear
  * Step-by-step
  * Logical

### 💡 Example

```python
print("Hello, World!")
```

### 📌 Important Points (MCQ)

* Programming ≠ typing → it is **problem solving**
* Computer has **no intelligence** → only executes instructions

### 🎯 Exam Focus

* Definition-based MCQs
* Identify correct examples of instructions
* Mistake: Thinking computer understands human language directly

# 🔹 12. Introduction to Loops

### ✅ Definition

* Loop = **repeat instructions**

### 🔑 Types

* For loop
* While loop

### 💡 Example (Python)

```python
for i in range(5):
    print(i)
```

### 📌 Important Points

* Avoid repetition
* Saves code length

### 🎯 Exam Focus

* Output prediction
* Loop counting questions

---

# 🔹 13. More about Loops

### 🔑 Concepts

* Nested loops
* Infinite loops

### 💡 Example

```python

    print("Hello")
```

### 📌 Important Points

* Infinite loop → runs forever
* Must have stopping condition

### 🎯 Exam Focus

* Identify infinite loops
* Loop termination conditions

---

# 🔹 14. Solution to Looping Problem

### ✅ Idea

* Use loops instead of repeated code

### 💡 Example

Without loop:

```python
print(1)
print(2)
print(3)
```

With loop:

```python
for i in range(1, 4):
    print(i)
```

### 🎯 Exam Focus

* Code optimization questions
