# 🎮 1. What is Scratch?

### 🧠 Simple Idea

Scratch is a **visual programming language** where you **drag and drop blocks** instead of typing code.

### 🧍 Real-Life Analogy

Think of it like **LEGO blocks**:

* Each block = one instruction
* Join blocks → program

### ✅ Why used?

* No syntax errors
* Easy for beginners
* Focus on **logic (VERY IMPORTANT for NPTEL)**

---

### ✅ Example

👉 “Make a character say Hello”

Steps:

1. Drag `when green flag clicked`
2. Drag `say "Hello"`

---

### 📝 Practice

👉 Make sprite say your name

---

### 🎯 Exam Focus

* Scratch = **block-based language**
* Used for **learning logic**
* ❗ MCQ: “Why Scratch is beginner-friendly?”

# 🧱 3. How to Give Instructions

### 🧠 Idea
